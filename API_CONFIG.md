# API sozlash

API base URL bitta joydan o'zgaradi: `src/config/api.js`

## O'zgartirish

1. **.env fayl** (tavsiya etiladi):
   - Loyiha ildizida `.env` yarating
   - `REACT_APP_API_URL=http://localhost:8000` qo'shing
   - Production: `REACT_APP_API_URL=https://your-api.com`

2. **To'g'ridan-to'g'ri** `src/config/api.js`:
   - `API_BASE_URL` qiymatini o'zgartiring
